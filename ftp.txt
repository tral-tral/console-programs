Randomness cannot exists, but since it cannot exist everything that happens is based on the cause and effects of the past. Thereof everything is pre-determined, but the very fact that everything is pre-determined is what makes it unpredictable. If you were to predict the future, you would be able to change, but knowing what could happen would another variable in the predictation of will happen, therefore being a different future. So if the future is different from each so pre-determined future, it will infinitely stack into possible predicted futures. The world can be pre-determined, but not predictable.

To check the user list.
extern 0 0 -2 shares
A file named ~ will be recieved that lists all the share directories.

To check the list of a user.
extern 0 0 -2 ^[user]
A file named ~ will be recieved that lists all the files in that share directory.

To download a file.
extern 0 0 -2 ~[user] [file]

To upload a file.
send [secretID] 0 [file] , File names cannot contain the character ~.

To remove a file
extern [secretID] 0 remove [file]
